Starter Code: Classes

CS1 – Processing

Use this starter code as a template for your class-based project. Copy and modify it to match your own object design.

Main Sketch (Tab 1)


// Declare your objects here
MyObject obj1;
MyObject obj2;
MyObject obj3;

void setup() {
  size(600, 400);

  // Create objects using the constructor
  obj1 = new MyObject(100, 200);
  obj2 = new MyObject(300, 150);
  obj3 = new MyObject(500, 250);
}

void draw() {
  background(255);

  obj1.move();
  obj1.display();

  obj2.move();
  obj2.display();

  obj3.move();
  obj3.display();
}

Class Definition (New Tab)

Create a new tab and name it the same as your class (for example: MyObject). 

class MyObject {

  // Instance variables (properties)
  float x, y;
  float speedX, speedY;
  float size;
  color col;

  // Constructor
  MyObject(float x, float y) {
    this.x = x;
    this.y = y;

    // Initialize other properties
    speedX = random(-3, 3);
    speedY = random(-3, 3);
    size = random(20, 50);
    col = color(random(255), random(255), random(255));
  }

  // Movement behavior
  void move() {
    x += speedX;
    y += speedY;
  }

  // Display behavior
  void display() {
    fill(col);
    ellipse(x, y, size, size);
  }

}

Optional: Add Boundaries


void checkBounds() {
  if (x < 0 || x > width) {
    speedX = -speedX;
  }
  if (y < 0 || y > height) {
    speedY = -speedY;
  }
}

Add this method and call it inside draw() if you want bouncing behavior.

Optional: Using an Array (Advanced)


MyObject[] objects = new MyObject[10];

void setup() {
  size(600, 400);

  for (int i = 0; i < objects.length; i++) {
    objects[i] = new MyObject(random(width), random(height));
  }
}

void draw() {
  background(255);

  for (MyObject obj : objects) {
    obj.move();
    obj.display();
  }
}

What to Modify

  • Rename MyObject to your class name
  • Add or change properties
  • Customize the drawing in display()
  • Add your own methods (e.g., rotate, grow, animate)

Common Mistakes

  • Forgetting to use new when creating objects
  • Putting void before the constructor (don’t do that!)
  • Forgetting to call methods in draw()
  • Mismatch between class name and tab name
