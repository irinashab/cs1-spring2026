Midterm Practice Sheet

CS1 Midterm Practice Sheet

This practice sheet mirrors the structure and difficulty of the midterm. You should be able to answer every question without running code.

1) Drawing and Coordinate Reasoning

a. What will this code draw? Be precise.

size(120,120);
line(0,height/2,width,height/2);
line(width/2,0,width/2,height);

b. Describe exactly what appears on the screen.

void setup(){
  size(200,200);
}

void draw(){
  fill(255);
  rect(0,0,width,height);
  fill(0);
  ellipse(width/2,height/2,100,100);
}

2) Translate Into English

int score = 85;

if(score >= 90){
  println("A");
}else{
  println("Not an A");
}

Is the condition true or false? What prints?

3) Short Concept Questions

  • What is camelCase? Give an example.
  • What is the difference between == and =?
  • What does x += 3; mean?
  • What is a variable?
  • Why do we need different data types? Give two examples.

4) Write Code to Produce the Image

  • 300x300 window
  • Blue background
  • White circle centered
  • Two small black circles as eyes

5) Randomness

Write a line of code that simulates rolling an 8-sided die (values 1–8 inclusive).

6) Boolean Logic

boolean ready = false;
println(!(!ready));

What prints? Explain.

7) Loops

a. Write a for loop that prints all odd numbers from 5 to 21 (including 21).

b. Rewrite the same logic using a while loop.

8) Scope and Variables

a. What is variable scope?

b. Will this compile? If yes, what prints? Explain.

int x = 10;

void setup(){
  int y = 5;
}

void draw(){
  println(x);
  println(y);
}

c. What are the values of a and b? Draw a memory diagram.

int a = 5;
a--;
a -= 2;
int b = a;

9) Functions

int f(int n){
  int total = 0;
  for(int i = 0; i <= n; i++){
    total += i;
  }
  return total;
}

What does the function return when n = 3? When n = 5? Describe in one sentence what it does.

boolean g(int n){
  return n > 0;
}

What does it return when n = -2? When n = 7? What does it check?

10) Arrays

int[] nums = {3,6,9,12,15};
  • What does nums[2] print?
  • What does nums[nums[0] / 3] print?
  • What does nums.length return?
  • Write one line that prints a random element from the array.

11) Mouse Conditions

  • Write a condition that checks if the mouse is on the left half of the screen.
  • Write a condition that checks if the mouse is in the top-right quadrant.

12) Tricky Loop

int i;
for(i = 2; i < 6; i++);
{
  println(i);
}

What prints? Explain carefully.

13) Challenging Array Tracing

int[] arr = {2,4,6,8,10,12};
println(arr[1]);
println(arr[arr[0]/2]);
println(arr[arr[2]/3]);

Now add:

println(arr[arr[3]]);

What happens?

How to Use This Practice Sheet

  • Answer without running code.
  • Explain compile vs run-time errors clearly.
  • Draw memory diagrams where appropriate.
  • Check loop bounds and array indices carefully.
