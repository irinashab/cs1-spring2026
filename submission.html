Programming Assignments: Naming & Submission Guidelines

When submitting a programming assignment, please ensure that you adhere to the following requirements:

1) File naming convention

  • Each programming sketch should be named using this format: firstInitialLastName_AssignmentName_Q#
  • Examples: iShablinsky_Lab1_Q2 or iShablinsky_HW1_Q1
  • The .pde extension will be added automatically by Processing, so you do not need to type it.
  • Allowed characters: letters, numbers, and underscores (_). Do not use spaces or other special characters.

2) Top-of-file comments

Every sketch must include comments at the very top that specify:

  • Your first and last name
  • Assignment name
  • Question number
  • Date the assignment was completed

Example:

// Alan Turing
// Homework 3
// Question 2
// 01/23/1937

3) How to submit

  1. Locate your sketch folder (Processing menu: Sketch → Show Sketch Folder).
  2. Compress (zip) the entire sketch folder.
  3. Upload the compressed (.zip) file to the assignment submission.
If you are submitting multiple sketches for a single assignment: place all sketch folders inside one parent folder, then compress (zip) that parent folder and submit the single .zip.
