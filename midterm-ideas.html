Project Ideas (Suggestions)

You are free to invent your own idea. The suggestions below are only meant to help you get started. Your program should demonstrate the use of variables, functions, loops, and arrays.

  • Bouncing Balls Playground
    Create a simulation with multiple balls bouncing around the screen. Use arrays to store positions and speeds, loops to update them, and functions to move and draw each ball.
  • Random Art Generator
    Create a program that generates a different geometric pattern each time it runs or when the user clicks the mouse. Experiment with colors, shapes, and symmetry.
  • Mini Drawing Program
    Let the user draw on the screen with the mouse. Use keys to change brush size or color, and store drawing points in arrays.
  • Reaction Time Game
    The screen changes color at random times and the user must click as quickly as possible. Measure reaction time and store previous scores.
  • Catch the Falling Objects
    Objects fall from the top of the screen and the player controls a paddle or basket to catch them. Keep track of score.
  • Animated Grid
    Create a grid of shapes that change color, size, or movement over time. Use nested loops to draw the grid and arrays to store properties.
  • Digital Aquarium
    Fish move across the screen at different speeds and directions. Arrays store fish positions and loops update their movement.

Note: These are only examples. Creative ideas are welcome.

Please do not submit Pong or Pong-style games. Pong is a classic beginner project, but we have seen it many times before. Try something a bit more original.
